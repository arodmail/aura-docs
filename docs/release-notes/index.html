Release Notes

AURA 1.1

New Features

  • Access Request Process - Defines a simple process for users and applications to request access to Aura
  • Migration API: Migrates Functional Credentials
  • Migration API: Enhances logging during migration and provides a summary log
  • Credentials Manager: Exposes Controlled Credentials

Fixes

  • Credentials API: Search should match only a full Device ID, not partial
  • Credentials API: Swagger documentation improvement

AURA 1.0

New Features & Components

  • Credentials Manager - A web application, provides check-in/check-out. Designed for SOC Analysts.
  • Credentials API - A microservice that exposes methods to manage credentials in a secure vault.
  • Migration API - A microservice that exposes methods to migrate credentials from Remedy to a secure vault.
  • User API - A personalization microservice that interfaces with BluePages to get user profile photos to display in Credentials Manager.
  • AURA Docs - This set of AURA system documentation
  • Secret Server - An easy-to-use, full-featured PAM solution from Thycotic.
